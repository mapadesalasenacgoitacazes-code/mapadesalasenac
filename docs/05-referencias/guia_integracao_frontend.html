Regra importante: A pasta models/ e a config/ ja estao prontas. O frontend trabalha SOMENTE na pasta views/ e nos arquivos de css/, js/ e img/.

2. Como Rodar o Projeto

Opcao A: XAMPP (recomendado)

1
Abra o XAMPP e inicie Apache e MySQL
2
Copie a pasta do projeto para C:\xampp\htdocs\mapa_de_salas\
3
Acesse no navegador:
http://localhost/mapa_de_salas/src/public/index.php

Opcao B: Servidor PHP embutido

1
Inicie o MySQL pelo XAMPP
2
No terminal, rode: 
cd C:\caminho\do\projeto\src\public
C:\xampp\php\php.exe -S localhost:8888
3
Acesse:
http://localhost:8888
Banco de dados: O banco mapa_de_salas ja foi criado no MySQL com dados de exemplo. Se precisar recriar, rode o arquivo database/schema.sql no phpMyAdmin.

Regra critica: Conflito de reserva

A tabela reserva tem constraint UNIQUE(id_sala, data, turno). Isso significa que nao e possivel reservar a mesma sala, no mesmo dia, no mesmo turno. O sistema verifica isso automaticamente antes de criar a reserva.

7. Como Editar as Views (Frontend)

Cada view PHP ja recebe as variaveis prontas do index.php. Voces so precisam usar essas variaveis no HTML.

Exemplo: Listar salas na view

<!-- As variaveis $salas, $andares ja vem do index.php -->

<?php foreach ($salas as $sala): ?>
  <tr>
    <td><?= htmlspecialchars($sala['numero']) ?></td>
    <td><?= htmlspecialchars($sala['tipo']) ?></td>
    <td><?= $sala['capacidade'] ?></td>
  </tr>
<?php endforeach; ?>

Exemplo: Mostrar dados do usuario logado

<!-- Dados da sessao disponiveis em qualquer view -->

<p>Ola, <?= $_SESSION['usuario_nome'] ?>!</p>
<p>Seu perfil: <?= $_SESSION['perfil'] ?></p>

Exemplo: Mostrar conteudo por perfil

<?php if (in_array($_SESSION['perfil'], ['Admin', 'Supervisao'])): ?>
  <button>Excluir</button> <!-- So Admin e Supervisao veem -->
<?php endif; ?>
Dica: Sempre use htmlspecialchars() ao exibir dados do banco para evitar XSS.

8. Como Funcionam os Formularios

Os formularios enviam dados via POST para o index.php com page e acao.

Exemplo: Formulario de cadastro de sala

<form method="POST" action="index.php?page=salas&acao=cadastrar">
  <input type="text" name="numero" required>
  <select name="tipo">
    <option value="Sala de Aula">Sala de Aula</option>
    <option value="Laboratorio">Laboratorio</option>
  </select>
  <input type="number" name="capacidade" required>
  <input type="text" name="andar">
  <button type="submit">Salvar</button>
</form>

Campos obrigatorios por formulario

FormularioCampos POST
Salanumero, tipo, capacidade, andar, observacao
Turmacodigo, id_curso, turno, data_inicio, data_termino_prevista, dias_semana[], professores[], professor_principal
Reservaid_sala, id_turma, data, turno, observacao
Cursonome, tipo_curso, area, programa, carga_horaria_total
Usuarioid_unidade, nome, email, senha, perfil

Mensagens de sucesso: Apos salvar, o sistema redireciona com ?msg=criada. O header.php exibe automaticamente o alerta verde de sucesso.
